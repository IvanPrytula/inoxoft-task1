## Django_DRF_React quick starter

#### Functionality
* [x] Basic Signup & Login/Logout with JWT
* [x] API documentation with DRF Swagger


### Quick Install

- Clone the repo

#### Backend
```
$ cd inoxoft-task1 
$ cd backend
$ pip3 install -r requirements.txt
$ ./manage.py makemigrations
$ ./manage.py migrate
$ ./manage.py runserver
```

#### Frontend
```
$ cd frontend
$ npm install
$ npm start
```
